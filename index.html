Technical Architecture

Our modular design separates key components to enable flexibility, reliability, and integration with blockchain networks.

SUI Connection Manager

Handles communication with the SUI blockchain, including transaction submission, state queries, and event monitoring.

Fallback Manager

Monitors blockchain connections and automatically switches to alternative sequencers if the primary connection fails.

Module Manager

Manages deployment and versioning of your code modules across blockchain networks, ensuring consistency.

Transaction Manager

Executes your code in our secure runtime environment, processes external data, and prepares transactions for the blockchain.

Enterprise-Grade Reliability

Our unique Fallback Manager ensures transactions are processed even during network disruptions.

How the Fallback Manager Works

  • 1
    Continuous Monitoring: The Fallback Manager constantly monitors the health and performance of connected blockchains.
  • 2
    Failure Detection: If SUI (primary sequencer) experiences delays, congestion, or outages, the system detects the issue immediately.
  • 3
    Automatic Switching: Transactions are automatically rerouted to alternative sequencers (Ethereum, Celestia, etc.) without any action required.
  • 4
    Transaction Integrity: The same transaction logic and data is preserved across chains, ensuring consistent execution.
  • 5
    Auto-Recovery: When the primary sequencer is operational again, the system can automatically return to it for future transactions.

    # Get current weather conditions

    if current_weather["conditions"] == "sunny":

DeFi Stop-Loss Order with Market Data
// Create ModRoll client with SUI connection
const client = new ModRollClient({
  connectionType: 'sui',
  network: 'mainnet',
  fallbackEnabled: true
});

// Define a stop-loss order function
async function createStopLossOrder(token, amount, stopPrice) {
  // Connect to price feed
  const priceFeed = new DataFeed('crypto_price', {
    token: token,
    source: 'binance'
  });
  
  // Get current token price
  const currentPrice = await priceFeed.getData();
  
  // Check if stop loss should trigger immediately
  if (currentPrice.price <= stopPrice) {
    return executeOrder(token, amount);
  }
  
  // Set up conditional transaction
  const transaction = new Transaction();
  
  // Add price condition
  transaction.addCondition({
    type: 'price_below',
    token: token,
    threshold: stopPrice,
    data_source: 'binance'
  });
  
  // Add sell order instruction
  transaction.addInstruction('market_sell', {
    token: token,
    amount: amount
  });
  
  // Deploy conditional transaction
  const result = await client.deployConditionalTransaction(transaction);
  
  return {
    status: 'pending',
    order_id: result.order_id,
    transaction_id: result.transaction_id
  };
}

Supply Chain Verification on SUI
module Product.SupplyChainVerifier {
    use Sui::object::{Self, UID};
    use Sui::transfer;
    use Sui::tx_context::{Self, TxContext};
    
    // Product struct representing a physical item
    struct Product has key, store {
        id: UID,
        name: String,
        manufacturer: address,
        manufacture_date: u64,
        batch_id: String,
        current_location: String,
        temperature_log: vector<u64>,
        verified: bool
    }
    
    // Update product location (called from ModRoll with IoT data)
    public fun update_location(
        product: &mut Product,
        new_location: String,
        current_temperature: u64,
        _ctx: &mut TxContext
    ) {
        product.current_location = new_location;
        vector::push_back(&mut product.temperature_log, current_temperature);
    }
    
    // Verify the product meets quality standards
    public fun verify_product(
        product: &mut Product,
        ctx: &mut TxContext
    ) {
        // Check temperature log for any values outside acceptable range
        let i = 0;
        let temp_ok = true;
        while (i < vector::length(&product.temperature_log)) {
            let temp = *vector::borrow(&product.temperature_log, i);
            if (temp < 2 || temp > 8) {  // Example: 2-8¬∞C range for pharmaceuticals
                temp_ok = false;
                break;
            };
            i = i + 1;
        };
        
        // Update product verification status
        product.verified = temp_ok;
    }
}
